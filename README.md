# mira-sample-s3-webhosting

Use `npm` with this package.

# Getting Started

1. Install all modules

   ```bash
   npm install
   ```

2. Build

   ```bash
   npm run build
   ```

3. Set up your config file

   ```bash
   cp config/default.sample.json config/default.json
   # edit this new default.json file e.g. update the prefix
   ```

4. Deploy

   ```bash
   npm run deploy
   ```
